## Course Description

CSCI 113 explores the fundamental concepts of **computer organization and architecture**, focusing on the internal structure and operation of modern computer systems.

Topics covered in this course include:

- Register-transfer level (RTL) computer design  
- Logical design of combinational circuits  
- Sequential circuit design  
- Arithmetic Logic Unit (ALU) organization  
- CPU architecture and design  
- Memory hierarchy and cache organization  
- Input/output system design  
- Hard-wired control units  
- Microprogrammed control  
- Simulation of computer organization  
- Introduction to high-performance superscalar processors  

The course combines **digital logic principles with computer architecture concepts** to understand how hardware components work together to execute programs.
